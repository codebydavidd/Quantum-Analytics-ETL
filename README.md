<h1 align="center"> Quantum Analytics ⚛️</h1>

## 💻 Sobre o projeto

Projeto de **Engenharia de Dados** focado na construção de um pipeline ETL (Extract, Transform, Load) automatizado. O objetivo é unificar, limpar e estruturar dados fragmentados de e-commerce (Olist) para criar uma base sólida e confiável para análises de Business Intelligence.

## 🤯 O sistema realiza:

- **Extração Automatizada:** Leitura dinâmica de múltiplos datasets brutos (Pedidos, Itens, Produtos, Pagamentos);
- **Unificação de Dados:** Cruzamento de tabelas relacionais (Merges) para criar uma visão completa da venda;
- **Tratamento de Dados:** Limpeza de valores nulos, correção de tipagem e padronização de categorias;
- **Engenharia de Atributos:** Criação de colunas calculadas para facilitar a análise temporal e financeira;
- **Exportação Otimizada:** Geração de um arquivo mestre (`.csv`) pronto para consumo em Power BI ou Tableau.
